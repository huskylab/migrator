# migrator
Однофайловый PHP-скрипт для надежной миграции WordPress. Идеален для больших баз данных: разбивает экспорт на части для предотвращения тайм-аутов. Автоматически и корректно заменяет домен, правильно обрабатывая сериализованные данные в настройках и виджетах. Простой пошаговый веб-интерфейс.
